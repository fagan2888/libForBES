Group LASSO problems

Group LASSO problems make use of the sum-of-norms regularization which is defined as

\[ g(x) = \mu \sum_{i=1}^{k} \|x_{(i)}\|_2, \]

where \(x_{(i)}\in\mathbb{R}^k\) is the \(i\)-th part of vector \(x\in\mathbb{R}^n\) of length \(k\).

It is assumed that \(x\) is of dimension \(n=mk\) for some \(m\in\mathbb{N}\).

The group-LASSO problem, in its simplest form, is given by

\[ \mathrm{minimize}\ \|z\|_2^2 + \mu \sum_{i=1}^{k} \|x_{(i)}\|_2, \]

subject to

\[ z = Ax - b. \]

Let us give an example using matrix \(A\in\mathbb{R}^{4\times 6}\)

\[ A = \begin{bmatrix} 1&-2&3&-4&5&3\\ 2&-1&0&-1&3&2\\ -1&0&4&-1&2&3\\ -1&-1&-1&1&3&5 \end{bmatrix} \]

and

\[ b= \begin{bmatrix}1&2&3&4\end{bmatrix}^{\top}, \]

with regularization parameter \(\mu=5.0\) and grouping parameter \(k=2\).

const size_t n = 6;
const size_t m = 4;
double data_A[] = {
1, 2, -1, -1,
-2, -1, 0, -1,
3, 0, 4, -1,
-4, -1, -3, 1,
5, 3, 2, 3,
3, 2, 3, 5 };
double data_minus_b[] = {-1, -2, -3, -4};
Matrix A(m, n, data_A);
Matrix minus_b(m, 1, data_minus_b);
LinearOperator * OpA = new MatrixOperator(A);
Function * f = new LogLogisticLoss();
double lambda = 5.0;
Function * g = new SumOfNorm2(lambda, 2);

In a very similar fashion as in the case of LASSO, we formulate and solve the problem as follows

FBProblem prob = FBProblem(*f, *OpA, minus_b, *g);
Matrix x0(n, 1);
double gamma = 0.999/A.norm_fro_sq();
FBSplitting * solver = new FBSplitting(prob, x0, gamma);
solver->run();
Matrix xstar = solver->getSolution();

The algorithm converges in \(25\) iterations to the following minimizer

\[ x^{\star}= \begin{bmatrix} 0 & 0 & 0 & 0 & 0.5069 & 0.638 \end{bmatrix}^{\top}. \]
