Detailed Description

Quadratic loss over an affine subspace

Version
version 0.1
Date
Created on November 3, 2015, 3:58 PM
Author
Pantelis Sopasakis

We define the quadratic loss over an affine subspace as the following function \(f:\mathbb{R}^n\to\mathbb{R}\cup \{+\infty\}\) given by

\[ f(x) = \frac{1}{2} \sum_{i=1}^{n}w_i (x_i - p_i)^2 + \delta(x\mid \mathcal{Z}), \]

where \(\mathcal{Z}\) is an affine subspace of \(\mathbb{R}^n\) given by

\[ \mathcal{Z} = \{z\in\mathbb{R}^n \mid A z = b\}. \]

For this function we define its conjugate \(f^*(y)\) and the gradient of its conjugate \(\nabla f^*(y)\).

We first define the matrix \(F\) as

\[ F = A \cdot \mathrm{diag}(1/\sqrt{w_i})_i, \]

and we perform once and store the LDL-factorization of \(FF^{\top} + \epsilon I\) for a small \(\epsilon>0\), that is

\[ FF^{\top} + \epsilon I = LDL^{\top}. \]

For a given \(y\) we define the vector \(\sigma=\sigma(y)\) as

\[ \sigma_i = \frac{y_i}{w_i} + p_i, \]

and let \(q=q(y)\) be the solution of the linear system

\[ (FF^{\top} + \epsilon I)q = A\sigma - b. \]

We solve this system using the above LDL-factorization.

Now the gradient of the conjugate function is given by

\[ \nabla f^*(y)_i = \sigma_i - \frac{(A^{\top}q)_i}{w_i}. \]

and \(f^*(y)\) is given by

\[ f^*(y) = y^{\top}\nabla f^*(y) - \frac{1}{2}\bar{g}^{\top}\mathrm{diag}(w_i)\bar{g}, \]

with \(\bar{g} = \nabla f^*(y) - p\).
