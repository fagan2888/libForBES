About LibForBES

ForBES is a solver for smooth and non-smooth convex optimization problems.

LibForBES is a C++ implementation of ForBES powered by the high-performance computational routines of BLAS, LAPACK and SuiteSparse.

LibForBES can be used to solve any convex problem using forward-backward splitting methods.

Examples of problems which can be solved with libfores include, but are not limited to

  • Linear and quadratic problems with polyhedral constraints
  • LASSO and group LASSO problems
  • Sparse log-logistic regression
  • Elastic net regularization problems
  • Optimal control problems
  • Problems involving loss functions (Hinge, Huber functions, etc)
  • Quadratically constrained problems
  • Basic pursuit problems
  • Support vector machine training
  • Problems involving the nuclear norm of a matrix

and many more.

Getting started

The main components of libforbes are the following classes:

  1. The Function API which provides access to a collection of functions
  2. The Linear Operator API which defines generic linear operators
  3. The Forward-backward problem where the specifications of an optimization problem are provided
  4. The Solver API with various solver implementations such as FBSplitting.

For a quick start check out our list of examples.

Installation

You can download libforbes from our github page and build it on your system.

Installing libforbes is very easy: just run the installation script install.sh.

If you are experiencing any trouble with the installation, contact the developers.

Read also our detailed installation guide.

You can then use libforbes in your own C++ project:

  1. Include the header file ForBES.h
  2. Link to the libforbes static library (libforbes.a)
