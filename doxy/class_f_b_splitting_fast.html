Detailed Description

Fast forward-backward splitting algorithm.

Version
version 0.0

FBSplittingFast extends FBSplitting by prepending the Nesterov extrapolation step to the iterate() method. On convex problems this is known to enforce the optimal convergence rate (for the objective value) of order \(\mathcal{O}(1/k^2)$\).
