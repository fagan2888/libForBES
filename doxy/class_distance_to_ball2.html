Detailed Description

Distance from the unit ball of R^n.

Version
version 0.1
Date
Created on January 14, 2016, 2:02 AM
Author
Pantelis Sopasakis

This class extends Function and implements the squared distance to the closed ball of norm-2 in \(\mathbb{R}^n\) of radius \(\rho\), centered at \(c\in\mathbb{R}^n\) which is defined by

\[ B_2(\rho, c) = \{x\in\mathbb{R}^n: \|x-c\|_2 \leq \rho\}, \]

that is, this is the function

\[ f(x) = \frac{w}{2}d(x, B_2(\rho, c))^2 = \frac{w}{2}\|x-\mathrm{proj}(x, B_2(\rho, c))\|^2, \]

where \(w>0\) is a positive scalar and

\[ \mathrm{proj}(x, B_2) = \mathrm{argmin}\limits_{\|y-c\|\leq \rho}\|y-x\| \]

is the projection of \(x\) on \(B_2(\rho, c)\). This is computed as

\[ \mathrm{proj}(x, B_2) = \begin{cases} x, &\text{ if } \|x-c\|\leq \rho\\ c+\frac{\rho}{\|x-c\|}(x-c), &\text{ otherwise} \end{cases} \]

Essentially, the squared distance to ball-2 is given by

\[ f(x) = \begin{cases} 0, &\text{ if } \|x-c\|\leq \rho,\\ \frac{w}{2}(\|x-c\|-\rho)^2,&\text{ otherwise} \end{cases} \]

The gradient of \(f\) is given by

\[ \begin{align} \nabla f (x) &= \frac{w}{2}(x-\mathrm{proj}(x, B_2(\rho, c)))\\ &= \begin{cases} 0, & \text{if } \|x-c\|\leq \rho,\\ \left(1-\frac{\rho}{\|x-c\|}\right)(x-c), &\text{otherwise} \end{cases} \end{align} \]
