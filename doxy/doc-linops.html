Linear Operators

Working with linear operators

The Linear Operator API

A linear operator \(T:\mathbb{R}^n \to \mathbb{R}^m\) may not always be available in the explicit form of a matrix, but in some implicit form to which we don't have direct access.

What is more, one may define linear operators over spaces of matrices, for instance \(T: X\to Y\), where either or both of \(X\) and \(Y\) can be \(\mathbb{R}^{n\times m}\).

Think of a linear operator T as a black box function for which we only know that it is linear and we can evaluate its value \(T(x)\) at a given vector x and, sometimes, the value \(T^*(x^*)\) of its adjoint operator (which is again defined implicitly).

You can extends the class LinearOperator to define your own linear operators. You will then need to implement the following methods:

virtual Matrix call(Matrix& x) = 0; // define T(x)
virtual Matrix callAdjoint(Matrix& x) = 0; // define T^*(y)
virtual bool isSelfAdjoint() = 0; // is it a self-adjoint operator?
virtual std::pair< size_t, size_t > dimensionIn() = 0; // x - dimension
virtual std::pair< size_t, size_t > dimensionOut() = 0; // T(x) - dimension

The input and output spaces' dimensions are defined as a pair of size_t where the first entry corresponds to the number of rows and the second entry is the number of columns.

This is an example of how a linear operator can be defined and used:

#include "ForBES.h"
int main(int argc, char** argv) {
size_t n = 10;
size_t m = 3;
Matrix M = MatrixFactory::MakeRandomMatrix(m, n, 0.0, 10.0, Matrix::MATRIX_DENSE);
Matrix x = MatrixFactory::MakeRandomMatrix(n, 1, 0.0, 10.0, Matrix::MATRIX_DENSE);
LinearOperator *T = new MatrixOperator(M);
Matrix y;
y = T->call(x); // y = T(x)
Matrix z = M*x; // z = M*x
// z and y are equal
std::cout << "Result, y = \n" << y;
std::cout << "\n\nz = \n" << z;
delete T;
}
/*
* Typical output:
*
* Result, y =
* Matrix 3x1
* Type: Dense
* 310.8
* 374.4
* 263.9
*
* z =
*
* Matrix 3x1
* Type: Dense
* 310.8
* 374.4
* 263.9
*
*/
