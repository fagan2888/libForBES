Detailed Description

Indicator of a second-order cone

Version
0.0
Author
Lorenzo Stella
Date
Created on September 21, 2015, 10:07 AM

This class implements the indicator function of second order cones (SOC). If \(n\) is the dimension of the considered space, then this is the set

\[ \mathrm{SOC}(n) = \left\{(x, t) \in \mathbb{R}^n : x \in \mathbb{R}^{n-1} \text{and } \|x\| \leq t\right\} \]

The dimension \(n\) must be provided to the constructor.

Ontological categorization of the function.

The category of a Function defines which methods are implemented, that is, it specifies whether one can compute

  1. \(f(x)\),
  2. \(f^*(x)\),
  3. \(\nabla f(x)\),
  4. \(\nabla f^*(x)\),
  5. \(\mathrm{prox}_{\gamma f}(v)\) and \(f(\mathrm{prox}_{\gamma f}(v))\)
