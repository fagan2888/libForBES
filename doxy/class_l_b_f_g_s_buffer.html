Detailed Description

Buffer for the Limited Memory BFGS algorithm.

Version
version 0.1

LBFGSBuffer is a finite length buffer for pairs \((s_k, y_k)\) which are necessary for the computation of \(r_k = H_k q_k\) in the LBFGS algorithm.

double LBFGSBuffer::hessian_estimate ( )

Computes an estimation of the Hessian using the buffered info.

The Hessian approximation is a scaled-identity matrix of the form \(H = \gamma I\); this method returns the scalar \(\gamma\).

For details see Equation (7.20) in J. Nocedal and S.J. Wright, Numerical Optimization, Second edition, Springer, 2006.

int LBFGSBuffer::update ( const Matrix *  q,
Matrix *  r,
double &  gamma0 
)

Perform the two-loop update and computes

\[ r_k = H_k q_k, \]

for a given vector \(q_k\) without explicitly forming or using \(H_k\).

Details about the algorithm can be found in J. Nocedal and S.J. Wright, Numerical Optimization, Second edition, Springer, 2006 (see Alg. 7.4).

Parameters
q(input) given vector \(q_k\)
r(output) result, that is \(r_k = H_k q_k\).
gamma0(input) Initial guess of the Hessian at the first iteration which is given by \(H_k^0 = \gamma_0^0 I\). A possible choice is computed by hessian_estimate.
Returns
Returns STATUS_OK on success
