Separable sum of functions

Separable sum properties

We often encounter functions which are written as a separable sum of other functions whose proximal operators is known. In such cases, one can invoke the separable sum property to compute the proximal operator of the composite function.

For \(x\in\mathbb{R}^n\) and \(\mathcal{I}\) be a set of unique indices in \(\mathbb{N}_{[1,n]}\) given by \(\mathcal{I}=\{i^1, i^2, \ldots, i^r\}\), define \(x_{\mathcal{I}} = (x_{i^1},\ldots, x_{i^r})\).

Take a function \(f:\mathbb{R}^n\to\mathbb{R}\) and sets of indices \(\mathcal{I}_1, \ldots, \mathcal{I}_2\) so that \(\mathcal{I}_i \cap \mathcal{I}_j = \emptyset\) whenever \(i \neq j\) so that

\[ f(x) = \sum_{i=1}^{s} f_i(x_{\mathcal{I}_j}). \]

Such a function is called a separable sum.

The proximal operator of a separable sum can be computed as follows:

\[ (\mathrm{prox}_{\gamma f}(v))_{\mathcal{I}_j} = \mathrm{prox}_{\gamma f_j}(v_{\mathcal{I}_j}). \]

As a result, it suffices to be able to compute \( \mathrm{prox}_{\gamma f_j}(\cdot)\) to compute the proximal of a separable sum.

The conjugate of a separable sum function is also easy to compute as

\[ f^*(y) = \sum_{i=1}^{s} f_i^*(y_{\mathcal{I}_j}). \]

Code snippet

Let us now give a code snippet to demonstrate how this is implemented in libforbes. Assume first our function is written as follows:

\[ f(x) = f_1(x_0, x_3, x_4) + f_2(x_1, x_2), \]

for \(x=(x_0, x_1, x_2, x_3, x_4)\in\mathbb{R}^5\). Here, we have \(\mathcal{I}_1=\{0,3,4\}\) and \(\mathcal{I}_2=\{1,2\}\).

Let us define these two sets of indices:

size_t num_functs = 2; /* 2 functions */
size_t num_idx_1 = 3; /* set I_1 contains 3 indices */
size_t num_idx_2 = 2; /* set I_2 contains 2 indices */
std::vector<size_t> idx1(num_idx_1);
std::vector<size_t> idx2(num_idx_2);
idx1[0] = 0;
idx1[1] = 3;
idx1[2] = 4; /* f1(x_0, x_3, x_4) */
idx2[0] = 1;
idx2[1] = 2; /* f2(x_1, x_2) */

Say, \(f_1\) is an ElasticNet function with parameters \(\lambda=2.5\) and \(\mu=1.3\) and \(f_2\) is a HingeLoss function with parameters \(b=(0.8, 1.5)\) and \(\mu=1.4\).

Function * f1 = new ElasticNet(2.5, 1.3);
Matrix b(2, 1);
b[0] = 0.8;
b[1] = 1.5;
Function * f2 = new HingeLoss(b, 1.5);

We can now map the above functions to their corresponding set of indices using a std::map<Function*, std::vector<size_t>* > object using which we can create a SeperableSum function:

std::map<Function*, std::vector<size_t>* > fun_idx_map;
fun_idx_map[f1] = &idx1;
fun_idx_map[f2] = &idx2;
Function * sep_sum = new SeparableSum(fun_idx_map);
