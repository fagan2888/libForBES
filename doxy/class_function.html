Detailed Description

A ForBES function.

Version
version 0.2
Date
Created on July 9, 2015, 3:35 AM
Author
Pantelis Sopasakis

This is a generic API for ForBES functions. Constructors for this class are protected. Instances of this class can be created using any of its subclasses a list of which can be found here.

ForBES functions are all assumed to follow be \(f:X \to \mathbb{R}\cup \{+\infty\}\), where \(X\) is a vector space, typically either \(\mathbb{R}^n\) or \(\mathbb{R}^{n\times m}\).

In general, a Function in libforbes is a tuple \((f(\cdot),\nabla f, f^*(\cdot), \nabla f^*(\cdot), \mathrm{prox}_{\gamma f}(\cdot), f(\mathrm{prox}_{\gamma f}(\cdot)))\) where some of these components are available and implemented.

Certain functions furhter allow Hessian-vector products, i.e., computation of terms of the form \(\langle \nabla^2 f(x), z\rangle\) and/or with the gradient of their conjugate \(\langle \nabla^2 f(x), z\rangle\).

Input and output arguments to Function calls are instances of Matrix.

For example, in Quadratic, which is a subclass of Function, the first four of these components are available, namely \((f(\cdot),\nabla f, f^*(\cdot), \nabla f^*(\cdot))\).

In ElasticNet, only the triplet \((f(\cdot),\mathrm{prox}_{\gamma f}(\cdot), f(\mathrm{prox}_{\gamma f}(\cdot)))\) is available.

virtual FunctionOntologicalClass Function::category ( )
pure virtual

Ontological categorization of the function.

The category of a Function defines which methods are implemented, that is, it specifies whether one can compute

  1. \(f(x)\),
  2. \(f^*(x)\),
  3. \(\nabla f(x)\),
  4. \(\nabla f^*(x)\),
  5. \(\mathrm{prox}_{\gamma f}(v)\) and \(f(\mathrm{prox}_{\gamma f}(v))\)
  6. \(\langle \nabla^2 f(x), z\rangle\) and
  7. \(\langle \nabla^2 f^*(x), z\rangle,\).
Returns
Ontological class of the current Function object.

Implemented in SumOfNorm2, ConjugateFunction, DistanceToBall2, SeparableSum, DistanceToBox, QuadraticLossOverAffine, IndBox, IndPos, Quadratic, QuadOverAffine, ElasticNet, QuadraticLoss, Norm1, Norm2, IndBall2, HingeLoss, IndProbSimplex, LogLogisticLoss, HuberLoss, Norm, IndSOC, and LQCost.
