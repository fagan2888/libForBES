int Norm::dualNorm ( Matrix &  x,
double &  norm 
)
virtual

The dual norm is defined as

\[ \|x\|_\star = \sup_{\|z\|\leq 1} \langle z, x\rangle. \]

Knowing the dual norm allows us to define the conjugate function of a norm, that is for \(f(x)=\|x\|\), its conjugate is given by

\[ f^*(y) = \begin{cases} 0,&\text{if } \|y\|_\star \leq 1,\\ \infty,&\text{otherwise} \end{cases} \]

The conjugate function of \(f(x)=\|x\|\) is the indicator function of the dual ball \(\mathcal{B}_\star = \{y\mid \|y\|_\star \leq 1 \}\).

Parameters
xpoint in \(\mathbb{R}^n\) where the dual norm should be computed
normthe value of the dual norm at x, that is \(\|x\|_\star\)
Returns
status code which is equal to STATUS_OK if the computation has succeeded without any problems, STATUS_UNDEFINED_FUNCTION if this function is not defined by the derived class and STATUS_NUMERICAL_PROBLEMS if some numerical problems prevented the computation of a reliable result. Custom implementations are allowed to return other non-zero error/warning status codes.
