Detailed Description

Euclidean norm.

Version
version 0.1
Date
Created on October 30, 2015, 6:54 PM
Author
Pantelis Sopasakis

This class defines the function

\[ f(x) = \mu\|x\|_2, \]

for vectors \(x\in\mathbb{R}^n\) and scalars \(\mu>0\). Function \(f\) is a norm \(\|x\| = f(x)\) for all \(\mu>0\) with dual

\[ \|x\|_* = \frac{1}{\mu}\|x\|_2. \]

This allows us to define the conjugate function of this function which is the indicator of the \(\|\cdot\|_*\)-ball, that is

\[ f^*(y) = \begin{cases} 0,&\text{if } \|y\|_\star \leq 1,\\ \infty,&\text{otherwise} \end{cases} \]

The proximal operator of this function is given by

\[ \mathrm{prox}_{\gamma f}(v) = \begin{cases} 0, &\text{ if } \|v\| \leq \gamma \mu,\\ (1-\frac{\gamma\mu}{\|v\|})v,&\text{ otherwise} \end{cases} \]

int Norm2::dualNorm ( Matrix &  x,
double &  norm 
)
virtual

The dual norm is defined as

\[ \|x\|_\star = \sup_{\|z\|\leq 1} \langle z, x\rangle. \]

Knowing the dual norm allows us to define the conjugate function of a norm, that is for \(f(x)=\|x\|\), its conjugate is given by

\[ f^*(y) = \begin{cases} 0,&\text{if } \|y\|_\star \leq 1,\\ \infty,&\text{otherwise} \end{cases} \]

The conjugate function of \(f(x)=\|x\|\) is the indicator function of the dual ball \(\mathcal{B}_\star = \{y\mid \|y\|_\star \leq 1 \}\).
