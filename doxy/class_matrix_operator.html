Calls the adjoint of this operator.

For an operator \(T:X \to Y\), its adjoint is defined as an operator \(T^*:Y \to X\) so that for all \(x\in X\) and every \(y Y\) it is

\[ \langle y, T(x) \rangle_{Y} = \langle T^*(y), x \rangle_{X} \]

For a matrix \(M\in\mathbb{R}^{m\times n}\) its adjoint is its transpose, that is \(M^* = M^{\top}\) and, indeed, for all \(x\in\mathbb{R}^n\) and \(y\in\mathbb{R}^m\) it is \(\langle y, Mx \rangle = y^{\top}Mx = (M^{\top}y)^{\top}x = \langle M^{\top}y, x \rangle = \langle M^*(y), x \rangle\).

This method updates a given vector or matrix \(y\) as

\[ y\leftarrow \gamma y + \alpha T^*(x), \]

where \(\alpha\) and \(\gamma\) are given constant scalars.

std::pair< size_t, size_t > MatrixOperator::dimensionIn ( )
virtual

For a linear operator \(T:X \to Y\), it returns the dimension of its domain.

The in-dimension of an operator \(f:\mathbb{R}^{n}\to Y\) is the pair (n,1).

The in-dimension of \(f:\mathbb{R}^{n\times m}\to Y\) is the pair (n,m).

In some cases, this dimension may be undefined, so then this method returns (0, ?); such a case may be a linear operator which reverses the order of the entries of x which may be applied to vectors of any size. In such a case, the in-dimension will be (0,1). It is, however, strongly recommended to define the nonzero operator's dimensions in all cases.
