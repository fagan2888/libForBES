Matrix factorizations and solvers

A matrix factorization is, in general, the procedure of decomposing a given matrix into a sum or product or other combination of two or more matrices which possess favourable properties which can be exploited for instance to solve a linear system.

The ForBES API offers a simple interface to factor-solve procedures exporting to the client only what is necessary: a factor and a solve step.

This is done by the abstract methods factorize() and solve() in FactoredSolver which are implemented by derived classes.

Matrix Q = ...; // A symmetric positive definite matrix
FactoredSolver * solver = new ...; // Some factored solver
solver->factorize(); // Factorize once
solver->solve(b, x); // Solve the system A*x = b
solver->solve(c, x); // Solve A*x = c
delete solver; // Release resources

Note that a FactoredSolver, is a class derived from the abstract class MatrixSolver which is an abstraction layer for solvers of linear systems given in the form \(Ax=b\), where \(A\) is a matrix.

A more general case is addressed by LinSysSolver which is a generic and abstract solver for linear systems given either in the above form or in the form \(T(x)=b\), where \(T\) is a linear operator (instance of LinearOperator).

Cholesky

The Cholesky factorization can be applied to any symmetric positive definite matrix.

A symmetric positive definite matrix \(A\) can be written as \(A=LL'\), where \(L\) is a lower-triangular matrix.

Using this matrix one can solve linear systems of the form \(Ax=b\) (for a given \(b\)) at the cost of a forward and a backward substitution.

LDL factorization

The LDL factorization (often referred to as \(LDL'\) can be applied to any symmetric matrix. A given symmetric matrix \(A\) is decomposed as \(A=LDL'\) where \(L\) is lower-triangular and \(D\) is a diagonal matrix.

This is evidently similar to the Cholesky factorization described above, but does not require that the factorized matrix be positive definite. LDL can be applied to any symmetric matrix.
