Detailed Description

Forward-backward splitting algorithm.

Version
version 0.0

FBSplitting specializes IterativeSolver by defining how the iterate() method works. In particular, it computes the next iterate as the forward-backward (or proximal gradient) step at the current point.

Examples:
qp_box.cpp.
