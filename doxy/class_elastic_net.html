Detailed Description

Elastic net regularization function.

Version
version 0.1
Date
Created on October 28, 2015, 7:43 PM
Author
Pantelis Sopasakis

This class extends the class Function and implements the elastic net regularization function which is a function \(g:\mathbb{R}^n\to\mathbb{R}_{+}\) defined as

\[ g(x) = \mu \|x\|_1 + \frac{\lambda}{2}\|x\|_2^2. \]

Its proximal operator is then defined as

\[ \mathrm{prox}_{\gamma g}(v) = \frac{t(v)}{1+\lambda \gamma}. \]

In order to define function \(t:\mathbb{R}^n\to\mathbb{R}\) we need first to introduce the following function \(\psi:\mathbb{R}^n\times\mathbb{R}^n \to \mathbb{R}^n\) as

\[ (\psi(z,v))_i = \begin{cases} z_1,&\text{ if } v_i>0\\ -z_i,&\text{ if } v_i>0\\ 0,&\text{ otherwise} \end{cases} \]

We also denote the element-wise absolute value of a vector \(x\in\mathbb{R}^n\) by \(|x|\) and we define the function

\[ \tau(v) = \max(0, |v|-\gamma\mu). \]

Then, function \(t\) is given by

\[ t(v) = \psi(\tau(v), v). \]

This class also allows the computation of the value of the function at a proximal point, that is \(g(\mathrm{prox}_{\gamma g}(v))\), as a function of \(v\).

This is given by

\[ g(\mathrm{prox}_{\gamma g}(v)) = \mu 1^{\top} \frac{t(v)}{1+\lambda \gamma} + \frac{\lambda}{2}\|\frac{t(v)}{1+\lambda \gamma}\|_2^2. \]

This a simple MATLAB implementation of this function:

function [prox, g] = elasticNetProx(x, gam, mu, lam)
uz = max(0, abs(x)-gam*mu)/(1+lam*gam);
prox = sign(x).*uz;
g = mu*sum(uz)+(0.5*lam)*(uz'*uz);

Ontological categorization of the function.
