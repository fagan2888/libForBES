Detailed Description

Conjugate of a given function.

Version
version 0.1
Date
Created on November 7, 2015, 3:15 AM
Author
Pantelis Sopasakis

This class is a utility which provides access to the convex conjugate of a function which is passed to the constructor of ConjugateFunction by reference.

For a given function \(s:X\to\mathbb{R}\cup \{+\infty\}\), this class provides access to \(f(x) = s^*(x)\). Then,

  1. Function \(f(x) = s^*(x)\)
  2. Gradient \(\nabla f(x) = \nabla s^*(x)\)
  3. Conjugate function \(f^*(x) = s(x)\)
  4. Conjugate gradient \(\nabla f^*(x) = \nabla s(x)\)
  5. Proximal operator \(\mathrm{prox}_{\gamma s^*}(v) = v - \mathrm{prox}_{\gamma s}(v)\)

See also the documentation of the various methods in this function

For a given function \(s\), provided in the constructor, this method computes the proximal gradient of its conjugate which is given by

\[ \mathrm{prox}_{\gamma s^*}(v) = v - \gamma \mathrm{prox}_{\gamma^{-1} s}(\gamma^{-1}v). \]
