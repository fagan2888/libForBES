Detailed Description

Low-level forward-backward operations management class.

Version
version 0.0

FBCache handles forward-backward operations related to an FBProblem In particular, objects of the FBCache class are initialized given an FBProblem p, a point x (of class Matrix) and a scalar parameter gamma. It allows to evaluate the proximal-gradient operation starting from x, and the value of the forward-backward envelope function (FBE) associated with p.

int FBCache::extrapolate_f ( double  tau,
double &  fxtd

protected

Computes \(f(x+\tau d)\) for the stored values of \(x\) and \(d\).

It uses the formula

\[ f(x+\tau d) = f_1(r_1(x+\tau d)) + \langle l, x\rangle + \tau \langle l, d\rangle + f_2(r_2(x) + \tau v), \]

where \(v=L_2 d\) and \(f_1(r_1(x+\tau d))\) is computed using extrapolate_f1.

Postcondition
After successful completion, the status of the cache will be at least STATUS_EVALF.

Computes \(f_1(r_1(x+\tau d))\) for the stored values of \(x\) and \(d\).

It makes use of the formula

\[ f_1(r_1(x+\tau d)) = f_1(r_1(x)) + \beta_1 \tau + \beta_2 \tau^2, \]

where \(\beta_1 = \langle u, (\nabla f_1)(r_1(x))\), \(\beta_2 = u'Qu/2\) and \(u=L_1 d\).

int FBCache::extrapolate_fbe ( double  tau,
double  gamma,
double &  fbe 
)

Computes the value of \(\varphi_\gamma(x+\tau d)\) for the cached values of \(x\) (using set_point) and \(d\) (using set_direction).

The extrapolation is based on the formula

\[ \varphi_\gamma(x+\tau d) = f(x+\tau d) + g(z(x+\tau d)) + \frac{1}{2\gamma}\|R_\gamma(x+\tau d)\|^2 - \langle \nabla f(x+\tau d), R_\gamma(x+\tau d) \rangle \]

where \(z(x+\tau d) = \mathrm{prox}_{\gamma g}(y(x+\tau d))\) and \(y(x+\tau d) = x + \tau d - \gamma \nabla f(x+\tau d)\) and \(R_\gamma(x+\tau d) = x+\tau d - z(x+\tau d)\).

The values of \(f(x+\tau d)\) and \(\nabla f(x+\tau d)\) are computed efficiently using extrapolate_f and extrapolate_gradf.

Precondition
it is necessary that you provide a direction \(d\) before you invoke this method.
Parameters
tauparameter \(\tau\)
gammaparameter \(\gamma\) of the FBE
fbe(output) value of \(\varphi_\gamma(x+\tau d)\)
Returns
status code: STATUS_OK on success, STATUS_CACHE_NO_DIRECTION if no direction is available.

void FBCache::reset ( int  status)
protected

Used to set the internal status of the object at a specific value. For example, if the point at which to evaluate operations is changed (using set_point) then the status is reset to STATUS_NONE; if gamma instead is changed, the status is reset to STATUS_EVALF. In fact, the value of f is independent of gamma, and is not to be recomputed.

int FBCache::update_eval_f ( bool  order_grad_f2)
protected

Evaluates \(f(x)\) and updates the internal status of FBCache.

Function \(f\) is evaluated as

\[ f(x) = f_1(L_1x + d_1) + f_2(L_2 x + d_2) + \langle l, x \rangle. \]

If the value of \(f\) has already been computed at this particular point \(x\), the the method returns STATUS_OK without re-computing this value as it has been cached.

Parameters
order_grad_f2whether the gradient of \(f_2\) should be computed alongside the value of this function. This may often be computationally more favourable compared to the separate computation of the value and the gradient.
Returns
status code (see ForBESUtils); returns a STATUS_OK if the computation was successful and sets the internal status of the cache to STATUS_EVALF.
int FBCache::update_eval_FBE ( double  gamma)
protected

Evaluates the FBE at x with parameter gamma, and updates the internal status.

The FBE is computed as

\[ \varphi_\gamma(x) = f(x) + g(\mathrm{prox}_{\gamma g}(x-\gamma \nabla f(x))) - \nabla f(x)^{\top}R_{\gamma}(x) + \frac{1}{2\gamma}\|R_\gamma(x)\|^2, \]

where

\[ R_\gamma(x) = x - \mathrm{prox}_{\gamma g}(x-\gamma \nabla f(x)), \]

is the fixed-point residual (FPR).

Returns
Status code, see ForBESUtils.

const int FBCache::STATUS_EVALF = 1
static

The following are available:

  • The residuals \( r_1(x)=L_1 x + d_1 \) and \( r_2(x)=L_2 x + d_2 \)
  • The product \( \langle l, x \rangle \)
  • The values \( f_1(r_1(x)) \) and \( f_2(r_2(x)) \)
  • The value of \( f(x) = f_1(r_1(x)) + f_2(r_2(x))\)
  • The gradient of \(f_1\) at the corresponding residual, \( \nabla f_1( r_1(x) ) \)
const int FBCache::STATUS_FBE = 4
static

The forward backward envelope \(\varphi_\gamma(x)\) has been computed and it is cached. Additionally, all data which correspond to STATUS_FORWARDBACKWARD have been computed and are available.

const int FBCache::STATUS_FORWARD = 2
static

The data which correspond to STATUS_EVALF are available and, additionally, the following are available

  • The gradient of \(f_2\) at the corresponding residual, \( \nabla f_2( r_2(x) ) \)
  • The gradient of \(f\) at \(x\), that is \(\nabla f(x)\)
  • The value of \(y = x - \gamma \nabla f(x)\)
const int FBCache::STATUS_FORWARDBACKWARD = 3
static

Everything that corresponds to STATUS_FORWARD (and STATUS_EVALF) has been computed and is cached and available and, additionally,

  • The proximal \(z=\mathrm{prox}_{\gamma g}(y)\) where \(y = x - \gamma \nabla f(x)\)
  • The value \(g(z)\)
  • The square norm of the fixed point residual, that is \(\|x-z\|^2\)
const int FBCache::STATUS_GRAD_FBE = 5
static

All information which has been computed up to level STATUS_FBE is available and, furthermore, the gradient of the FBE, that is \(\nabla \varphi_\gamma(x)\) has also been computed and is cached.

const int FBCache::STATUS_NONE = 0
static

The internal state of FBCache is stale, or it has not been updated.
