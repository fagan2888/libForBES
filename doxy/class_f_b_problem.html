Detailed Description

Low level FB problem description.

Version
version 0.0

FBProblem is a low-level call which provides access to the specifications of a generic optimization problem in the following form:

\[ \min_{x\in X} f_1(L_1 x + d_1) + f_2(L_2 x + d_2) + \langle l,x \rangle + g(x), \]

where \(X\) is either \(\mathbb{R}^n\) or \(\mathbb{R}^{n\times m}\) and \(f_1\) is a smooth quadratic function, \(f_2\) is a smooth function, \(L_1, L_2\) are linear operators and \(d_1, d_2\) are elements of the respective target spaces, \(l\in X\) and \(g\) is a closed, proper, convex function over X.
