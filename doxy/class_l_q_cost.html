FunctionOntologicalClass LQCost::category ( )
virtual

Ontological categorization of the function.

The category of a Function defines which methods are implemented, that is, it specifies whether one can compute

  1. \(f(x)\),
  2. \(f^*(x)\),
  3. \(\nabla f(x)\),
  4. \(\nabla f^*(x)\),
  5. \(\mathrm{prox}_{\gamma f}(v)\) and \(f(\mathrm{prox}_{\gamma f}(v))\)
  6. \(\langle \nabla^2 f(x), z\rangle\) and
  7. \(\langle \nabla^2 f^*(x), z\rangle,\).
Returns
Ontological class of the current Function object.
