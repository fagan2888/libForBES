QP Problems

Quadratic programming with examples

Here we will present various formulations of QP problems:

  • Problems with box constraints
  • with general inequality constraints
  • with both inequality and equality constraints
  • quadratically constrained problems
Box-constrained QP

Let us here see how to formulate and solve a QP using libforbes.

First, define the cost function as

\[ f(x) = \frac{1}{2}x^{\top}Qx + q^{\top}x. \]

Take for instance \(Q\in\mathbb{R}^{4\times 4}\) and \(q\in\mathbb{R}^{4}\) to be

\[ Q = \begin{bmatrix} 7& 2& -2& -1\\2& 3& 0& -1\\-2& 0& 3& -1\\-1& -1& -1& 1 \end{bmatrix}, \]

and

\[ q = \begin{bmatrix} 1&2&3&4 \end{bmatrix}^{\top}. \]

We may define function \(f\) as follows

const size_t n = 4;
double data_Q[] = {
7, 2, -2, -1,
2, 3, 0, -1,
-2, 0, 3, -1,
-1, -1, -1, 1 };
double data_q[] = {1, 2, 3, 4};
Matrix Q(n, n, data_Q);
Matrix q(n, 1, data_q);
Function * f = new Quadratic(Q, q);

Let us define the constraints using an indicator function as follows

\[ g(z) = \delta(z\mid C), \text{ where } C=\{z: -1\leq z\leq 1\}. \]

We can now simply define function \(g\) as follows

double lb = -1;
double ub = +1;
Function * g = new IndBox(lb, ub);

Let us now define and solve the problem:

\[ \mathrm{minimize}\ f(x) + g(x). \]

Here is an example:

// Define the forward-backward problem specifications
FBProblem prob(*f, *g);
// Introduce a stopping criterion:
const double rel_tolerance = 1e-3;
FBStoppingRelative sc(rel_tolerance);
// Choose an initial guess
Matrix x0(n, 1);
// Construct an instance of a FB solver
double gamma = 0.1;
const int maxit = 100;
FBSplitting * solver = new FBSplitting(prob, x0, gamma, sc, maxit);
// Run the solver and get the minimizer
int solver_status = solver->run();
Matrix xstar = solver->getSolution();

We chose \(\gamma=0.1\) because \(1/\|Q\|_2 \simeq 0.1163 > \gamma\).

In the end, do not forget to delete the various objects that were constructed.

delete solver;
delete f;
delete g;

The algorithm terminates in 15 iterations (this is returned by solver->getIt()) and the optimizer is

\[ x^\star = \begin{bmatrix}-0.3529&-0.7647&-1&1\end{bmatrix}^{\top} \]

and the optimal value is \(f(x^\star)+g(x^\star)=-7.853\).

The Fast FBS method can be also used as follows

FBSplittingFast * solver = new FBSplittingFast(prob, x0, gamma, sc, maxit);

which converges within the required accuracy (specified by the termination criterion) in 11 iterations.

Inequality constrained QP

Let us now solve step-by-step an inequality constrained problem which has the following form

\[ \mathrm{minimize}\ \frac{1}{2}x^{\top}Qx + q^{\top} x + \delta(Hx\mid C), \]

where \(Q\) is a symmetric positive definite matrix, \(H:\mathbb{R}^n\to\mathbb{R}^s\) is a linear operator and \(C\) is a set which is easy to project on; here, let us take

\[ C = \{z \in \mathbb{R}^s: -1 \leq z \leq 1\}. \]

We will use the following values of Q and q:

\[ Q = \begin{bmatrix} 3& 1& 1& 0\\1& 3& 0& -1\\1& 0& 3& 1\\0& -1& -1& 5 \end{bmatrix}, \]

and

\[ q = \begin{bmatrix} 1&2&3&4 \end{bmatrix}^{\top}. \]

const size_t n = 4;
double data_Q[] = {
3, 1, 1, 0,
1, 3, 0, -1,
1, 0, 3, 1,
0, -1, 1, 5};
double data_q[] = {1, 2, 3, 4};
Matrix Q(n, n, data_Q);
Matrix q(n, 1, data_q);

Using the splitting \(f(x) = \frac{1}{2}x^{\top}Qx + q^{\top}x\), \(g(z) = \delta(z\mid C)\) and \(z=Hx\), we need to formulate and solve the dual optimization problem which is simply

\[ \mathrm{minimize}\ f^*(-H^* y) + g^*(y), \]

for the dual variable \(y\in\mathbb{R}^s\).

We may now define functions \(f\), \(f^*\), \(g\) and \(g^*\) and the linear operator \(H:\mathbb{R}^4\to\mathbb{R}\)

\[ H = \begin{bmatrix} 1&1&1&1 \end{bmatrix}. \]

// Define f and f*
Quadratic f(Q, q);
ConjugateFunction f_conj(f);
// Define g and g*
double lb = -1;
double ub = +1;
IndBox g(lb,ub);
ConjugateFunction g_conj(g);
double data_H[] = {1, 1, 1, 1};
Matrix H(1, n, data_H);
H.transpose(); H *= -1.0;
MatrixOperator Op_Htr_minus(H);

We may now define and solve the FB problem as follows

FBProblem prob(f_conj, Op_Htr_minus, g_conj);
Matrix y0(1, 1);
FBStoppingRelative stop(1e-5);
FBSplittingFast fbs(prob, y0, 0.2, stop);
fbs.run();

and we obtain the dual and primal optimal solutions

Matrix ystar = fbs.getSolution();
Matrix H_ystar = H*ystar;
double f_conj_star;
Matrix xstar(n, 1);
f_conj.call(H_ystar, f_conj_star, xstar);

The algorithm converges in \(34\) iterations and yields the optimizer

\[ x^\star = \begin{bmatrix}0.5&-0.5&-0.5&-0.5\end{bmatrix}^{\top}. \]

Inequality and Equality Constrained QP

Let us study the case where where we have both equality and inequality constraints. The problem can then be written as

\[ \mathrm{minimize}\ \frac{1}{2}x^{\top}Qx + q^{\top}x + \delta(x\mid E) + \delta(Hx \mid C), \]

where \(E\) is an affine space given by \(E = \{x: Ax = b\}\) and \(C\) is a box.

We may then define

\[ f(x) = \frac{1}{2}x^{\top}Qx + q^{\top}x + \delta(x\mid E), \]

which is a quadratic function over an affine space and

\[ g(z) = \delta(z\mid C), \]

where \(z=Hx\).

In this example we define

\[ A = \begin{bmatrix}1&1&0&0\\0&0&1&1\end{bmatrix}, \]

and

\[ b = \begin{bmatrix}0.1\\0.2\end{bmatrix}. \]

And \(H\), \(Q\) and \(q\) will be as in the previous example where we had only inequality constraints.

The only difference from the previous example is that here we need to use the function QuadOverAffine.

const size_t p = 2;
double data_A[] = {
1, 0,
1, 0,
0, 1,
0, 1 };
double data_b[] = {
0.1,
0.2 };
Matrix A(p, n, data_A);
Matrix b(p, 1, data_b);
// Define f and f*
QuadOverAffine f(Q, q, A, b);
ConjugateFunction f_conj(f);
