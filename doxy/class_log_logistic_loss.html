Detailed Description

Log-logistic loss function.

Version
version 0.1
Date
Created on October 29, 2015, 5:08 PM
Author
Pantelis Sopasakis

This class extends the class Function and implements the log-logistic loss function which is a function \(f:\mathbb{R}^n\to\mathbb{R}\) defined as

\[ f(x) = \mu \sum_{i=1}^{n} -\ln \sigma(x_i), \]

where

\[ \sigma(z) = \frac{e^{z}}{1+e^{z}}. \]

The gradient of \(f\), \(\nabla f:\mathbb{R}^n \to \mathbb{R}^n \) is computed element-wise by

\[ \nabla f(x)_i = \mu ( \sigma(x_i) - 1 ). \]

The Hessian of this function is a diagonal matrix with diagonal elements

\[ (\nabla^2 f(x))_{ii} = \frac{\mu e^{x_i}}{(1+e^{x_i})^2}. \]

int LogLogisticLoss::hessianProduct ( Matrix &  x,
Matrix &  z,
Matrix &  Hz 
)
virtual

Method hessianProduct computes the inner product \(\langle \nabla^2 f(x), z\rangle,\) between a vector \(z\) and the Hessian of \(f\) computed at a point \(x\).

If this method is not implemented in the derived class of Function, then an approximation is return based on the finite difference scheme

\[ \langle \nabla^2 f(x), z\rangle \simeq \frac{1}{\epsilon}[\nabla f(x+\epsilon z) - \nabla f(x)], \]

where \(\epsilon=10^{-8}\) (a small tolerance).

If the gradient of \(f\) is not available, or it fails to be computed, then, an appropriate status code will be returned.
