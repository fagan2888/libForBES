Log-logistic Regression

Sparse Log-Logistic regression problems

\(\ell_1\)-regularized

Log-logistic loss is a popular loss function used widely in classification problems in machine learning and statistics.

The log-logistic loss function is given by

\[ L_\mu(x) = \mu \sum_{i=1}^{n} -\ln \sigma(x_i), \]

where \(\sigma(\tau)=e^{\tau}/(1+e^{\tau})\).

The log-logistic \(\ell_1\)-regularized optimization problem is formulated as follows

\[ \mathrm{minimize}\ \ L_{1.0}(Ax-b) + \lambda \|x\|_1, \]

where \(\lambda>0\) is a regularization parameter.

The term \(\lambda\|x\|_1\) is used to enforce sparsity in \(x\).

Let us see how one can formulate such an optimization problem and solve it using libforbes using

\[ A = \begin{bmatrix} 1& -2& 3& -4& 5\\ 2& -1& 0& -1& 3\\ -1& 0& 4& -3& 2\\ -1& -1& -1& 1& 3 \end{bmatrix} \]

and

\[ b = \begin{bmatrix} 1&2&3&4 \end{bmatrix}^\top \]

const size_t n = 5;
const size_t m = 4;
double data_A[] = {
1, 2, -1, -1,
-2, -1, 0, -1,
3, 0, 4, -1,
-4, -1, -3, 1,
5, 3, 2, 3 };
double data_minus_b[] = {-1, -2, -3, -4};
Matrix A(m, n, data_A);
Matrix minus_b(m, 1, data_minus_b);

We should then define the problem's functions and linear operator \(A:\mathbb{R}^5 \to \mathbb{R}^4\)

LinearOperator * OpA = new MatrixOperator(A);
Function * f = new LogLogisticLoss();
double lambda = 3.50;
Function * g = new Norm1(lambda);

It is now straightfoward to construct our optimization problem and solve it as follows

FBProblem prob = FBProblem(*f, *OpA, minus_b, *g);
// Specify an initial guess
Matrix x0(n, 1);
// Specify the parameter gamma
double gamma = 0.1;
// Construct a new instance of a solver (e.g., FBSplitting or FBSplittingFast)
FBSplittingFast * solver = new FBSplittingFast(prob, x0, gamma);
// Run the solver and get the solution
solver->run();
Matrix xstar = solver->getSolution();

The minimizer of the above problem is then found to be

\[ x^\star = \begin{bmatrix} 0& 0& 0& 0& 1.228\end{bmatrix}^\top, \]

in \(29\) iterations.

\(\ell_2\)–regularized

Similarly, we may formulate a L2-regularized logistic regression problem.

\[ \mathrm{minimize}\ \ \sum_{i=1}^{n} -\ln \sigma(z_i) + \frac{\lambda}{2} \|x\|_2, \]

subject to \(z=Ax-b\).

The only necessary modification in the above code is

Function * g = new Norm2(lambda/2.0);

for \(\lambda=6.5\) and using the fast variant of FBSplitting, FBSplittingFast, we compute the minimizer

\[ x^\star = \begin{bmatrix} -0.2037&-0.304&0.2162&-0.1751&1.135\end{bmatrix}^{\top} \]

in \(53\) iterations, whereas, FBSplitting converges to the solution in \(41\) iterations.
