Detailed Description

Indicator of a box.

Version
version 0.1
Date
Created on July 26, 2015, 5:22 PM
Author
Pantelis Sopasakis

This class extends the class Function and implements the indicator function of a box.

A box is a set of the form \(B_{[l,u]}=\{x \in \mathbb{R}^n: l\leq x \leq u\}\) and its indicator is defined as

\[ \delta (x\mid B_{[l,u]}) = \begin{cases} 0, \text{ if } x\in B_{[l,u]},\\ \infty, \text{ otherwise} \end{cases} \]

The proximal operator of this function is defined as

\[ \mathrm{prox}_{\gamma\delta(\cdot|B_{[l,u]})}(v) = \mathrm{mid}(v; l, u) = \min (\max(v, l), u) \]

These two functions are implemented and are available through call and callProx.

The conjugate of the indicator function of a box is the support function of that box, that is

\[ \begin{align} \delta^*(y\mid B_{[l,u]}) &= \sup_{x\in B_{[l,u]}} \langle y, x\rangle = \sup_{l \leq x \leq u} \langle y, x\rangle\\ &= \sup_{l \leq x \leq u} \sum_{i=1}^{n}y_ix_i\\ &= \sum_{i=1}^{n} \sup_{l_i \leq x_i \leq u_i} y_ix_i\\ &= \sum_{i=1}^{n} \max \{ y_i l, y_i u \} \end{align} \]

This function is available though callConj.

FunctionOntologicalClass IndBox::category ( )
virtual

Ontological categorization of the function.

The category of a Function defines which methods are implemented, that is, it specifies whether one can compute

  1. \(f(x)\),
  2. \(f^*(x)\),
  3. \(\nabla f(x)\),
  4. \(\nabla f^*(x)\),
  5. \(\mathrm{prox}_{\gamma f}(v)\) and \(f(\mathrm{prox}_{\gamma f}(v))\)
  6. \(\langle \nabla^2 f(x), z\rangle\) and
  7. \(\langle \nabla^2 f^*(x), z\rangle,\).
Returns
Ontological class of the current Function object.
