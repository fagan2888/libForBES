Log-logistic loss

The log-logistic loss function is a function \(f:\mathbb{R}^n\to \mathbb{R}\) defined as follows.

For \(x\in\mathbb{R}^n\) let \(e(x)\) be a vector whose \(i\)-th entry is equal to \(e^{x_i}\), where \(x_i\) is the \(i\)-th element of \(x\). Let

\[ (p(x))_i = \frac{(e(x))_i}{1+(e(x))_i} \]

where \((p(x))_i\) and \((e(x))_i\) stand for the \(i\)-th entries in \(p(x)\) and \(e(x)\) respectively.

Then, the log-logistic function with parameter \(\mu\) is defined as

\[ f(x; \mu) = -\mu \sum_{i=1}^{n}\log((p(x))_i) \]

The gradient of \(f\) is then

\[ (\nabla f(x; \mu))_i = \mu((p(x))_i-1) \]

and its Hessian is

\[ (\nabla^2 f(x; \mu))_i = \mathrm{diag} \{ \mu \frac{(e(x))_i}{[(e(x))_i + 1]^2} \} \]

Hingle loss

The class HingeLoss extends the class Function and implements the Hinge loss function which is a function \(f:\mathbb{R}^n\to\mathbb{R}\) defined as

\[ f(x) = \mu \sum_{i=1}^{n} \max(0, 1-Bx), \]

where \(B=\mathrm{diag}(b)\) is a diagonal matrix - where \(b\in\mathbb{R}^n\).

Note that in the above expression, \(z=Bx\in\mathbb{R}^n\) is a vector and \(1-z\) is the vector whose i-th entry is \(1-z_i\). What is more, \(\max(0, 1-Bx)\) is also a vector whose i-th entry is \(\max(0, 1-b_ix_i)\).

The proximal operator of the Hinge loss function is given by

\[ \mathrm{prox}_{\gamma f}(v)_i = \begin{cases} b_i \min(1, b_i + \gamma \mu), &\text{if } b_i x_i < 1\\ x_i, &\text{otherwise} \end{cases} \]

Huber loss

HuberLoss extends the class Function and implements the Huber loss function which is a function \(f:\mathbb{R}^n\to\mathbb{R}\) defined as

\[ f(x) = \sum_{i=1}^{n} l_{\delta}(x_i), \]

where

\[ l_{\delta}(x_i) = \begin{cases} \frac{1}{2\delta}\|x_i\|^2,&\text{if } |x_i|\leq \delta \\ |x_i| - \frac{\delta}{2},&\text{otherwise} \end{cases} \]

The gradient of the Huber loss function is given by

\[ \nabla f(x)_i = \begin{cases} \frac{|x_i|}{\delta},&\text{if } |x_i|\leq \delta \\ \mathrm{sign}(x_i),&\text{otherwise} \end{cases} \]

Quadratic loss

The class QuadraticLoss extends the class Function and implements the quadratic loss function which is a function \(f:\mathbb{R}^n\to\mathbb{R}\) defined as

\[ f(x) = \frac{1}{2} \sum_{i=1}^{n} w_i(x_i - p_i)^2, \]

where \(w,p\in\mathbb{R}^n\) are given vectors.

The conjugate of this function is differentiable and its gradient is

\[ (\nabla f^*(y))_i = p_i + \frac{y_i}{w_i}. \]

while the conjugate is computed as

\[ f^*(y) = \frac{1}{2} y' (\nabla f^*(y) + p). \]
