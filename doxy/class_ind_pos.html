Detailed Description

Indicator of the positive orthant

Version
0.1
Author
Pantelis Sopasakis
Date
Created on July 26, 2015, 4:36 PM

This class implements the indicator function of a translated positive orthant, that is the set

\[ K_+^l = \{x\in\mathbb{R}^n, x\geq l\} \]

Then, the indicator function of \(K^l_+\) is defined as

\[ \delta(x\mid K^l_+) = \begin{cases} 0, & \text{ if } x\in K_+^l,\\ +\infty, & \text{ otherwise} \end{cases} \]

The proximal operator of this indicator function is then

\[ \mathrm{prox}_{\gamma \delta(\cdot\mid K^l_+)}(v) = \max(v,l), \]

and the conjugate function is

\[ \begin{align} \delta^*(y\mid K^l_+) &= \sup_{x\geq l} \langle x, y \rangle\\ &= \sup_{x\geq l} \sum_{i=1}^{n} y_i x_i\\ &= \sum_{i=1}^{n} \sup_{x_i\geq l_i} y_i x_i, \end{align} \]

where

\[ \sup_{x_i\geq l_i} y_i x_i = \begin{cases} +\infty, &\text{ if } y_i > 0,\\ y_i l_i, &\text{ otherwise} \end{cases} \]

These functions are implemented as call, #callProx and #callConj respectively.
